    Class EventBus

    Central event bus for module communication using pub/sub pattern. Provides event subscription, publishing, and history management for decoupled architecture.

    Subcategory

    Infrastructure

    Example

    const eventBus = EventBus.getInstance();

    // Subscribe to events
    const subscriptionId = eventBus.subscribe('user:login', (payload) => {
      console.log('User logged in:', payload.data.userId);
    });

    // Publish events
    await eventBus.publish({
      source: 'auth',
      type: 'user:login',
      data: { userId: '123', timestamp: Date.now() }
    });

    // Unsubscribe when done
    eventBus.unsubscribe(subscriptionId);
