# Top 100 Classical Composers & 300 Practice Pieces by Grade

## Top 100 Classical Music Composers (Piano & Classical Guitar)

### Tier 1: Most Essential Composers (1-20)

1. **Johann Sebastian Bach** (1685-1750) - Baroque master, essential for both piano and guitar
2. **Ludwig van Beethoven** (1770-1827) - Revolutionary composer, bridged Classical to Romantic
3. **Wolfgang Amadeus Mozart** (1756-1791) - Classical period genius, prolific in all forms
4. **Frédéric Chopin** (1810-1849) - The "poet of the piano", master of Romantic piano music
5. **Franz Schubert** (1797-1828) - Melodic genius, master of Lieder and piano works
6. **Johannes Brahms** (1833-1897) - Romantic traditionalist, complex harmonies
7. **Claude Debussy** (1862-1918) - Impressionist pioneer, revolutionary harmonies
8. **Robert Schumann** (1810-1856) - Romantic composer, piano cycles and symphonies
9. **Franz Liszt** (1811-1886) - Virtuoso pianist-composer, transcendental études
10. **Sergei Rachmaninoff** (1873-1943) - Last great Romantic, demanding piano concertos
11. **Pyotr Ilyich Tchaikovsky** (1840-1893) - Russian Romantic, ballets and concertos
12. **Joseph Haydn** (1732-1809) - "Father of the Symphony", Classical period master
13. **Domenico Scarlatti** (1685-1757) - Baroque keyboard virtuoso, 555 sonatas
14. **Maurice Ravel** (1875-1937) - French impressionist, orchestral colors
15. **Alexander Scriabin** (1872-1915) - Russian mystic, innovative harmonies
16. **Béla Bartók** (1881-1945) - Hungarian modernist, folk influences
17. **Sergei Prokofiev** (1891-1953) - Russian modernist, percussive piano style
18. **Erik Satie** (1866-1925) - French eccentric, minimalist precursor
19. **Dmitri Shostakovich** (1906-1975) - Soviet era master, 24 Preludes and Fugues
20. **George Gershwin** (1898-1937) - American composer, jazz-classical fusion

## Notes on Usage

### For Autocomplete Implementation

- Store composers with full names and common variations
- Include nationality and era for context
- For pieces, store with:
  - Full title
  - Composer name
  - Grade level
  - Instrument (piano/guitar)
  - Genre (sonata, étude, etc.)
  - Key/Opus number where applicable

### Grade Level Considerations

- Grades 1-3: Beginner (0-2 years)
- Grades 4-6: Intermediate (2-5 years)
- Grades 7-8: Advanced (5-10 years)
- Grades 9-10: Professional/Concert level

### Database Schema Suggestions

```sql
CREATE TABLE composers (
  id INTEGER PRIMARY KEY,
  full_name TEXT NOT NULL,
  last_name TEXT NOT NULL,
  common_name TEXT,
  birth_year INTEGER,
  death_year INTEGER,
  nationality TEXT,
  era TEXT,
  primary_instrument TEXT
);

CREATE TABLE pieces (
  id INTEGER PRIMARY KEY,
  title TEXT NOT NULL,
  composer_id INTEGER,
  grade_level INTEGER,
  instrument TEXT,
  opus_number TEXT,
  key_signature TEXT,
  genre TEXT,
  FOREIGN KEY (composer_id) REFERENCES composers(id)
);
```
