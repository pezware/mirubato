# Glossary

Status: ✅ Active (expand as needed)

- Durable Object: Cloudflare primitive providing single-threaded, stateful coordination per key
- D1: Cloudflare SQLite-compatible database service used by API/services
- KV: Cloudflare key-value storage for cached or non-transactional data
- R2: Cloudflare object storage used for PDF and asset storage
- Sync: Real-time replication of user data between client and backend via WS + HTTP
- Idempotency Key: Client-generated key ensuring safe retry of write operations
- Repertoire: User’s set of pieces with progress and stats
- Scorebook: Sheet music library, organization, and viewing experience
- Logbook: Practice entries and counters tracked over time
